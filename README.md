# Chethan K - Portfolio Website

A personal portfolio website built using **React (Vite)** and **Tailwind CSS** to showcase my projects, skills, and experience as a Full Stack Developer. The site is responsive, fast, and simple to customize.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- Node.js
- Git & GitHub

## Features
- Responsive and mobile-friendly design
- Smooth scrolling and simple animations
- Sections for About, Skills, Projects, and Contact
- Easy to update and deploy on hosting platforms

## Steps to Run Locally
### Prerequisites
Ensure **Node.js** and **npm** are installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/chethu-2004-chethu/portfolio-chethan-k.git
2. Move into the project folder:
   cd portfolio-chethan-k

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open in your browser:
   http://localhost:5173
